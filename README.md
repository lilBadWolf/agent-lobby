## What You Can Do

- Real-time lobby chat with live presence and typing indicators (MQTT)
- Private direct messages (webRTC tunnel via STUN/ICE)
- One-to-one audio and video calls (maybe)
- Peer-to-peer file transfer in direct messages (webRTC)
- mp3/m4a/mp4 player - agentAMP

## Quick Start

1. Enter your handle
2. Click `INITIALIZE LINK`
3. Start chatting in the lobby
4. Open settings (`⚙`) anytime to tune audio, devices, themes, and DM effects
5. ... if you make sure you're connected... the writings on the wall
