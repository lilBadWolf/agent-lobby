# Release Notes

🎉 **Release highlights for agent-lobby 0.2.8**

- ✅ **AgentAMP metadata editor**: Edit track metadata directly from the playlist UI for faster organization.
- 🎛️ **Sortable playlists and metadata polish**: Playlists now support drag sorting, plus metadata tooltips, track names, and visible durations.
- 🧩 **Playback and UI improvements**: Fixed "Now Playing" when repeat-one is active, improved controls padding, and made pinned URLs more reliable.

> This patch sharpens the AgentAMP experience with richer track metadata and cleaner playlist playback behavior.
